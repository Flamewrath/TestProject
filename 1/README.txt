Для обеспечения требования сквозного пространства для объектов любых типов была создана дополнительная сущность.
	Сущность содержит атрибут, типа BLOB, который может содержать объекты любых типов.
	Для связи с основными атрибутами(документами) используется дата записи и тип документа.
