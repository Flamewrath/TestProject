1. Программа состоит мз двух блоков: библиотеки и интерфейса и написана с применением .NET 5.0.
	Бибилиотека кроссплаторменна, прекрасно компилируется на Unix и Windows.
	Служит только для полуения данных.
	Интерфейс разработан для десктопа (WPF Windows), получает данные с помощью библиотеки и выводит на форму.
2. Используется база данных MSSQL и для корректной работы на другой машине потреуется переписать строку подключения (хранится в ресурсах проекта).
